Advanced user capability editing, specifically for bbPress

== Description ==

A handy tool for viewing, editing, or resetting a specific user's forum capabilities.

== Installation ==

1. Place the 'bbp-capabilities' folder in your '/wp-content/plugins/' directory.
2. Visit your 'Plugins' page, and activate bbPress Advanced Capabilities.
3. Edit a user with a role in your forums.
4. Scroll to the bottom and you will see a new table, where you can edit or reset a user's forum capabilities.
